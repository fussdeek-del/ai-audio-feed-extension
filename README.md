# Audio-feed-extension 🔊

A browser extension that reads your X (Twitter) feed out loud using ElevenLabs turn your timeline into a podcast.

Free and open source. No subscriptions. Just plug in your ElevenLabs API key and go.

---

## Why

Long-form posts on X are painful to read. This extension converts them to audio using ElevenLabs voices — so you can listen while doing other stuff.

---

## Setup

1. Clone the repo
2. Go to `chrome://extensions` → enable Developer Mode → Load Unpacked → select the folder
3. Open the extension → paste your [ElevenLabs API key](https://elevenlabs.io)
4. Browse X — posts will be readable with one click

---

## Features

- Reads X posts aloud using ElevenLabs TTS
- Works with any ElevenLabs voice
- Free to use — you only pay ElevenLabs if you exceed their free tier
- Lightweight, no backend

---

## Stack

- JavaScript (vanilla)
- ElevenLabs API
- Chrome Extension Manifest V3
